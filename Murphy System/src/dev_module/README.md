# Dev Module

The `dev_module` package provides developer-facing utilities within the
Murphy System: code generation, scaffolding, and developer tooling APIs.

## Key Modules

| Module | Purpose |
|--------|---------|
| `dev_manager.py` | `DevManager` — manages developer workspace operations |
| `models.py` | `DevProject`, `Scaffold`, `GeneratedCode` models |
| `api.py` | FastAPI router for developer tooling endpoints |
