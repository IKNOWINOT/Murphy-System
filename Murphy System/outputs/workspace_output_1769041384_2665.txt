Librarian System (/Librarian)\n\n### 3.1 Purpose\n\nThe Librarian is an intelligent, context-aware assistant that:\n- Interprets your natural language intent\n- Maps your needs to system capabilities\n- Provides step-by-step guidance\n- Answers queries from other system components\n\n### 3.2 Core Workflow\n\n```\n1. User describes goal in natural language\n   ↓\n2. Librarian analyzes problem domain and context\n   ↓\n3. Librarian identifies relevant system modules/capabilities\n   ↓\n4. Librarian maps capabilities to user needs\n   ↓\n5. Librarian provides step-by-step guidance with commands\n   ↓\n6. User executes commands and provides feedback\n   ↓\n7. Librarian learns and improves recommendations\n```\n\n### 3.3 Knowledge Base Structure\n\n```python\nknowledge_base = {\n    "capabilities": {\n        "document_management": {\n    "description": "Execute parallel agent collectives for complex tasks",\n            "commands": ["/swarm create", "/swarm status", "/swarm results"],\n            "use_cases": ["Analyze complex "commands": [...],\n            "estimated_time": "30 minutes"\n        }\n    }\n}\n```\n\n### 3.4 Interface with Terminal/Orchestrator\n\n```python\nclass LibrarianInterface:\n    def handle_query(self, query: str, context: dict) -> LibrarianResponse:\n        """\n        Process user query and return guidance\n        \n        Args:\n            query: User\'s natural language question\n            context: Current system state and conversation history\n            \n        Returns:\n            LibrarianResponse with recommendations and commands\n        """\n        intent = self.analyze_intent(query, context)\n        capabilities = self.map_to_capabilities(intent, context)\n        guidance = self.generate_guidance(capabilities, context)\n        return LibrarianResponse(\n            intent=intent,\n            capabilities=capabilities,\n            guidance=guidance,\n            commands=guidance.commands,\n            confidence=guidance.confidence\n        )\n```\n\n### 3.5 Confidence Scoring\n\n```python\ndef calculate_confidence(query, intent, capabilities):\n    """\n    Calculate confidence in recommendation\n    \n    Factors:\n    - Intent clarity (0-1)\n    - Capability match score (0-1)\n    - Context relevance (0-1)\n    - Historical success rate (0-1)\n    \n    Returns:\n        Confidence score (0.0-1.0)\n    """\n    intent_score = self.measure_intent_clarity(query)\n    capability_score = self.score_capability_match(intent, capabilities)\n    context_score = self.assess_context_relevance(intent, context)\n    success_score = self.get_historical_success(intent, capabilities)\n    \n    weights = [0.3, 0.3, 0.2, 0.2]\n    confidence = sum(w * s for w, s in zip(weights, [\n        intent_score, capability_score, context_score, success_score\n    ]))\n    \n    return confidence\n```\n\n### 3.6 Fallback Behaviors\n\nWhen no clear guidance is Terminal/Orchestrator\n\n### 5.1 Purpose\n\nThe central command interpreter and routing system that processes all user inputs and coordinates system components. It\'s the interface layer between you (the human) and the automated systems.\n\n### 5.2 Natural Language Processing Pipeline\n\n```\nUser Input\n    │\n    ├── Intent Classification\n    │   └── What is user trying to do?\n    │\n    ├── Entity Extraction\n    │   └── What entities (documents, agents, plans) are mentioned?\n    │\n    ├── Context Resolution\n    │   └── What\'s the current state? Intent classification\n        intent = self.classify_intent(input_text)\n        \n        # Entity extraction\n        entities = self.extract_entities(input_text)\n        \n        # Context resolution\n        resolved_context = self.resolve_context(entities, context)\n        \n        # Ambiguity detection\n        ambiguities = self.detect_ambiguities(intent, entities, resolved_context)\n        \n        # Confidence scoring\n        confidence = self.calculate_confidence(intent, entities, ambiguities)\n        \n        return ProcessedInput(\n            intent=intent,\n            entities=entities,\n            context=resolved_context,\n            ambiguities=ambiguities,\n            confidence=confidence\n        )\n```\n\n---\n\n### 5.3 Command Interpretation\n\n**Parse Structure Commands:**\n```python\ndef parse_command(command_text: str) -> ParsedCommand:\n    """\n    Parse structured command\n    """\n    parts = command_text.split()\n    \n    if not parts:\n        raise InvalidCommandError("Empty command")\n    \n    command = parts[0]\n    args = parts[1:]\n    \n    # Factor 1: Intent clarity\n    intent_clarity = measure_intent_clarity(intent)\n    \n    # Factor 2: Entity completeness\n    entity_completeness = measure_entity_completeness(entities, intent)\n    \n    # Factor 3: Context match\n    context_match = assess_context_match(intent, context)\n    \n    # Factor 4: Pattern match\n    pattern_match = assess_pattern_match(intent, entities)\n    \n    # Factor 5: Historical success\n    historical_success = get_historical_success(intent, context)\n    \n    # Weighted score\n    confidence = (\n        0.30 * intent_clarity +\n        0.25 * entity_completeness +\n        0.20 * context_match +\n        0.15 * pattern_match +\n        0.10 * historical_success\n    )\n    \n    return confidence\n```\n\n**Thresholds:**\n\n| Confidence Range | Action | Example |\n|----------------|--------|---------|\n| > 0.85 | Auto-execute | `/status` - Show system metrics |\n| 0.60 - 0.85 | Confirm with user | `create proposal for Acme` - Confirm client and details |\n| < 0.60 | Request clarification | `fix the thing` - Ask "what thing?" |\n\n**UI Display:**\n\n```\nmurphy> create proposal for client\n\n⚠ Medium confidence (0.72): Client name not specified\n\nInterpretation:\n- Create proposal\n- Client: [unknown]\n\nDid you mean:\n1. Retrieval\n\n**Search:**\n```python\ndef search_presets(query: str, filters: dict = None) -> List[Preset]:\n    """\n    Search presets by query and filters\n    """\n    results = []\n    \n    # Keyword search\n    keywords = extract_keywords(query)\n    for keyword in keywords:\n        if keyword in index:\n            for preset_id in index[keyword]:\n                preset = presets[preset_id]\n                results.append(preset)\n    \n    # Apply filters\n    if filters:\n        results = [p for p in results if matches_filters(p, filters)]\n    \n    # Rank by relevance\n    ranked = rank_results(results, query)\n    \n    return ranked\n```\n\n**Recommendation:**\n```python\ndef recommend_presets(context: dict) -> List[Preset]:\n    """\n    Recommend presets based on context\n    """\n    # Get current state\n    state = get_system_state(context)\n    \n    # Find similar contexts\n    similar_contexts = find_similar_contexts(state)\n    \n    # Get presets used in similar contexts\n    preset_usage = {}\n    for ctx in similar_contexts:\n        for preset in ctx.presets_used:\n            preset_usage[preset.preset_id] = preset_usage.get(preset.preset_id, 0) + 1\n    \n    # Rank by usage and success rate\n    recommendations = []\n    for preset_id, count in preset_usage.items():\n        preset = presets[preset_id]\n        score = count * preset.success_rate\n        recommendations.append((preset, score))\n    \n    # Sort by score\n    recommendations.sort(key=lambda x: x[1], reverse=True)\n    \n    return [p for p, s in recommendations[:10]]\n```\n\n---\n\n#### 5. Resolve variable\n            var_name = param_value[2:-2]\n            bound_parameters[param_name] = context.get(var_name, param_value)\n        else:\n            bound_parameters[param_name] = param_value\n    \n    return BoundStep(\n        step_id=step.step_id,\n        action=step.action,\n        parameters=bound_parameters\n    )\n```\n\n**Permission Verification:**\n```python\ndef verify_permissions(preset: Preset, user: User) -> PermissionResult:\n    """\n    Verify user has required permissions\n    """\n    for permission in preset.required_permissions:\n        if not user.has_permission(permission):\n            return PermissionResult(\n                granted=False,\n                missing=permission\n            )\n    \n    return PermissionResult(granted=True)\n```\n\n**Execution Monitoring:**\n```python\ndef execute_preset(preset: Preset, context: dict) -> ExecutionResult:\n    """\n    Execute preset with monitoring\n    """\n    # Bind parameters\n    bound_preset = bind_parameters(preset, context)\n    \n    # Verify permissions\n    permission_check = verify_permissions(preset, current_user)\n    if not permission_check.granted:\n        raise PermissionDeniedError(permission_check.missing)\n    \n    # Execute steps\n    execution = Execution(preset=bound_preset)\n    for step in bound_preset.steps:\n        result = execute_step(step)\n        execution.add_result(result)\n        \n        if not result.success:\n            execution.add_error(result.error)\n            break\n    \n    # Log execution\n    log_execution(execution)\n    \n    return execution\n```\n\n---\n\n#### 6.
