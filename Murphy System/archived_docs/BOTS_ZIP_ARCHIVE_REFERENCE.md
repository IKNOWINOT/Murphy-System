# bots.zip Archive Reference

This archive reference captures items from `Murphy System/bots.zip` that should be retained for historical context during Murphy 3.0 migration.

## Items to archive (reference only)

- Nested `bots/bots.zip` inside the outer archive (duplicate of the TypeScript bot packages).
- Legacy Python bot implementations (root-level `.py` files containing `bot` in their name).
- `composite_registry.json` (legacy registry/config for the Python bots).

For the full, labeled inventory of all contents (including what to keep active for Murphy 3.0), see `Murphy System/BOTS_ZIP_INVENTORY_MURPHY_3.md`.
