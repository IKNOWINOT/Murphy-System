## Installation & Setup

### Quick Start
```bash
# Extract the package
unzip murphy_system_runtime.zip

# Run setup script
python setup_agentic_system.py

# Run demo
python demo_agentic_system.py

# Run interactive mode
python demo_agentic_system.py --interactive
```

### Programmatic Setup
```python
import sys
sys.path.insert(0, 'src')

from agentic_workflow_system import quick_setup

# Create and initialize the system
system = quick_setup()

# Talk to the system naturally
response = system.process_natural_language("set up the system")
print(response.message)

# Build a workflow
response = system.build_workflow("Analyze customer data")
print(response.message)

# Get system status
response = system.process_natural_language("/status")
print(response.data)
```
