# Starting the Murphy Integrated System

## Quick Start Guide

### 1. Start the Backend Server

```bash
cd murphy_integrated
python murphy_complete_backend_extended.py
```

The server will start on **http://localhost:6666**

### 2. Open the UI

Open your browser and navigate to:
```
http://localhost:6666/murphy_ui_integrated.html
```

Or open the file directly:
```bash
# On Linux/Mac
open murphy_ui_integrated.html

# On Windows
start murphy_ui_integrated.html
```
