## New API Endpoints (7 total)

1. **POST /api/pipeline/explode**
   - Input: Vague request like "Automate my business"
   - Output: Complete plan with org chart, tasks, confidence levels
   - Librarian generates 80%, identifies human input needs

2. **POST /api/pipeline/org-chart**
   - Input: Business description + public strategies
   - Output: Matched org chart, adapted strategies, Murphy implementation

3. **POST /api/pipeline/block/verify**
   - Input: Block + action (magnify/simplify/solidify)
   - Output: Recommendations, possibly split/absorb actions

4. **POST /api/pipeline/block/update**
   - Input: Block ID + new content
   - Output: Affected blocks, regeneration order

5. **POST /api/pipeline/info-source**
   - Input: Information needed + user capability
   - Output: Decision (user/generate/external) + cost if external

6. **POST /api/pipeline/schedule**
   - Input: List of blocks with dependencies
   - Output: Optimal execution order

7. **GET /api/pipeline/status**
   - Output: Pipeline status, registered blocks, timeline events

## How It Works

### Example: "Automate my publishing business"

```
1. User calls /api/pipeline/explode
   Request: "Automate my publishing business"
   
2. Librarian explodes request:
   - Org Chart: CEO, Research, Content, Editor, Marketing, Sales
   - Task Blocks: 15 blocks with dependencies
   - Confidence Levels: 12 GREEN, 2 YELLOW, 1 RED
   - Auto-generated: 80%
   - Human input needed: Brand guidelines (RED), Target audience (YELLOW)

3. User provides RED/YELLOW information
   
4. System calls /api/pipeline/schedule
   - Orders tasks by priority and dependencies
   - Execution order: Research → Content → Edit → Marketing → Sales

5. Each block has [Magnify] [Simplify] [Solidify] buttons
   - User clicks [Magnify] on "Content Creation"
   - System splits into 3 sub-agents: Writer, Researcher, Formatter

6. User manually edits "Research" block
   - System calls /api/pipeline/block/update
   - Shows: "Content, Marketing, Sales need regeneration"
   - User approves → Cascade regeneration

7. System generates complete automation
   - All blocks GREEN
   - Ready to deploy
   - Self-managing
```
