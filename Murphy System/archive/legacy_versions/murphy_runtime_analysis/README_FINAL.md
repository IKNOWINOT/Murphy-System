## 📊 System Overview

### What is Murphy System Runtime?

The Murphy System Runtime is a comprehensive **AI orchestration framework** that provides:

- **Organizational Modeling**: Define and manage complex organizational structures
- **Workflow Planning**: Create and execute complex workflows with governance
- **Task Execution**: Execute tasks with dependencies and retry logic
- **Code Generation**: Generate functional code in 10+ languages
- **Document Generation**: Generate PDF, Word, and HTML documents
- **Learning & Adaptation**: Track performance and adapt decisions
- **Autonomous Operation**: Priority-based scheduling and risk management
- **Dynamic Commands**: Automatic command discovery based on module coupling
- **State Management**: Comprehensive state tracking and persistence
- **Decision Making**: Rule-based and adaptive decision engines

## 🎯 Quick Start

### Installation

```bash
# Extract the package
unzip murphy_system_runtime.zip
cd murphy_system_runtime

# Install dependencies
pip install -r requirements.txt

# Install the package
pip install -e .
```

### Basic Usage

```python
from src.system_integrator import SystemIntegrator

# Initialize the system
system = SystemIntegrator()

# Check system status
status = system.get_system_status()
print(f"System Status: {status}")

# Execute a task
task = {
    "task_id": "task_001",
    "task_type": "analysis",
    "parameters": {"data": "sample data"}
}
result = system.execute_task(task)
print(f"Result: {result}")
```

### Using the Web Interface

```bash
# Start the web server
python -m http.server 8080

# Open in browser
# http://localhost:8080/terminal.html
```

## 🔧 Configuration

### System Configuration

```python
from src.system_integrator import SystemIntegrator

# Initialize with configuration
system = SystemIntegrator(
    enable_governance=True,
    governance_mode="BALANCED",
    enable_learning=True,
    enable_autonomy=True,
    log_level="INFO"
)
```

### Governance Modes

- **HEAVY**: Human-in-the-loop for everything
- **BALANCED**: Human-in-loop for high-risk, auto for low-risk
- **LIGHT**: Auto-execution at high confidence
- **AUTONOMOUS**: Full auto-execution
