## Installation Steps

### Windows (5 minutes):
1. Extract `murphy_system_complete_v1.0_windows.zip`
2. Run `install.bat`
3. Edit `groq_keys.txt` - add your API keys
4. Run `start_murphy.bat`
5. Open http://localhost:3002

### Linux/Mac (5 minutes):
1. Extract the ZIP
2. Run `chmod +x install.sh && ./install.sh`
3. Edit `groq_keys.txt` - add your API keys
4. Run `./start_murphy.sh`
5. Open http://localhost:3002
