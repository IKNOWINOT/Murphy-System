## Model: Open-Core

Murphy System follows an **open-core** business model — the same strategy
used by GitLab, Elastic, Hashicorp, and Confluent.  The core automation
engine, all connectors, and the testing framework are open-source under
the MIT license.  Enterprise features (multi-tenant SaaS, managed hosting,
premium support, white-label) are offered commercially.

## What Is Open-Source vs. Commercial

### Open-Source (Community Edition)

- All source modules in `src/`
- Runtime, gate synthesis, swarm orchestration
- All connectors (building, energy, manufacturing, enterprise, messaging, streaming, remote access)
- ML strategy engine, digital asset generator, content creator platforms
- Rosetta Stone heartbeat, agentic API provisioner, UI testing framework
- Terminal UIs and dashboards
- Test suites

### Commercial (Professional / Enterprise)

- Multi-tenant isolation and SaaS hosting
- Priority support and SLA agreements
- White-label branding
- Custom module development by InonI engineering
- On-premise deployment tooling and air-gapped packages
- SOC2 / HIPAA compliance audit assistance
- Advanced analytics with custom KPI dashboards
- Executive planning engine enterprise configuration
