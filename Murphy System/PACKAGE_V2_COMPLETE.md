## 🚀 Installation Instructions

### Windows (Recommended)

1. **Extract the ZIP file**
   ```
   Right-click murphy_system_v2.0_UI_UPDATED_20260130_1810.zip
   Select "Extract All..."
   Choose destination folder
   ```

2. **Run the installer**
   ```
   Double-click install.bat
   Wait for installation to complete
   ```

3. **Add API Keys**
   ```
   Open groq_keys.txt in Notepad
   Add your Groq API keys (one per line)
   Save and close
   ```

4. **Start Murphy**
   ```
   Double-click start_murphy.bat
   Wait for "Running on http://127.0.0.1:3002"
   ```

5. **Open the UI**
   ```
   Open your web browser
   Go to: http://localhost:3002
   ```

### Linux/Mac

1. **Extract the ZIP file**
   ```bash
   unzip murphy_system_v2.0_UI_UPDATED_20260130_1810.zip
   cd murphy_system
   ```

2. **Run the installer**
   ```bash
   chmod +x install.sh
   ./install.sh
   ```

3. **Add API Keys**
   ```bash
   nano groq_keys.txt
   # Add your keys, one per line
   # Save with Ctrl+X, Y, Enter
   ```

4. **Start Murphy**
   ```bash
   chmod +x start_murphy.sh
   ./start_murphy.sh
   ```

5. **Open the UI**
   ```
   Open your web browser
   Go to: http://localhost:3002
   ```

## 📞 Troubleshooting

### Issue: Port 3002 already in use
**Solution:** 
```bash
# Windows
netstat -ano | findstr :3002
taskkill /PID <PID> /F

# Linux/Mac
lsof -ti:3002 | xargs kill -9
```

### Issue: Messages stacking
**Solution:** Refresh the browser (F5)

### Issue: Scrolling not working
**Solution:** Check browser console for errors, try different browser

### Issue: Commands not executing
**Solution:** Verify backend is running on port 3002

### Issue: Socket.IO disconnects
**Solution:** Backend may have restarted, refresh page
