What R366 Wires (Next Round)

The wizard above shows the flow that ACTUALLY EXECUTED for Apex Wellness this round, run via CLI from the operator session. R366 ships:
  • POST /api/wizard/tenant — accepts the Step 1 form (legal name, industry, contact)
  • POST /api/wizard/roster — accepts multipart Excel file → calls r360 importer
  • POST /api/wizard/form — accepts multipart form file → calls r363+r364 pipeline
  • GET /api/wizard/status/<slug> — returns wizard progress for UI polling
  • Escalation/SLA — if compliance HITL stale >N days, bump up the org chart
