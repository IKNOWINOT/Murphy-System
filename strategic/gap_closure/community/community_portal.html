// QUICK START

Up and running in minutes

Three steps to deploy your first autonomous agent and start extending the Murphy System with custom connectors.

STEP 01 — INSTALL
Install the package
# requires Python 3.10+ pip install murphy-system # verify installation murphy --version
STEP 02 — INITIALIZE
Create your first agent
from murphy_system import MurphyAgent agent = MurphyAgent( name="my-agent", model="gpt-4o", auto_approve=False ) agent.run("Summarize open tickets")
STEP 03 — EXTEND
Build a custom connector
from connectors.plugin_sdk import ConnectorPlugin class MyPlugin(ConnectorPlugin): name = "my-plugin" version = "1.0.0" def execute(self, ctx): # your logic here return ctx.respond("done")
