# EQ (Everquest Integration)

The `eq` package provides experimental integration with EverQuest-inspired
game mechanics used for cultural-identity design and agent behavioural
training scenarios.

## Key Modules

| Module | Purpose |
|--------|---------|
| `agent_voice.py` | Assigns a culturally-aware voice/tone to each agent |
| `card_system.py` | Card-based action system for scenario simulation |
| `cultural_identity.py` | Models cultural-identity attributes per agent |
| `duel_controller.py` | Manages adversarial agent-vs-agent evaluation scenarios |
| `eq_game_connector.py` | Bridge to external EQ-compatible game APIs |
