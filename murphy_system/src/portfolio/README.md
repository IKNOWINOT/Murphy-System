# Portfolio

The `portfolio` package provides project portfolio management with
Gantt charts, critical-path analysis, and dependency tracking.

## Key Modules

| Module | Purpose |
|--------|---------|
| `gantt.py` | Gantt chart data model and rendering helpers |
| `critical_path.py` | Critical-path-method (CPM) calculation |
| `dependencies.py` | Project dependency graph management |
| `models.py` | `Project`, `Milestone`, `ResourceAllocation` models |
| `api.py` | FastAPI router for portfolio management |
