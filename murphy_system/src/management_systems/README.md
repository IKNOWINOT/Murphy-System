# Management Systems

The `management_systems` package provides high-level management automation
including recipe-based workflow templates, board generation, and form builders.

## Key Modules

| Module | Purpose |
|--------|---------|
| `automation_recipes.py` | Pre-built automation recipes for common management tasks |
| `board_engine.py` | Dynamic project-board generation from task templates |
| `dashboard_generator.py` | Auto-generates operational dashboards from module metadata |
| `doc_manager.py` | Manages living documentation for management processes |
| `form_builder.py` | Drag-and-drop form schema builder |
