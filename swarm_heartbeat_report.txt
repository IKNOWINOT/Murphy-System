The swarm heartbeat executor is responsible for sending a status signal to all 9 agents every 5 minutes. The executor uses the swarm_heartbeat function, which dispatches a signal to each domain and records a bus event. The domains included are: collector, translator, scheduler, executor, auditor, exec_admin, prod_ops, hitl, and rosetta. The signal_id is generated using uuid and the intent_hint is set to 'Swarm heartbeat — <domain>'.
