## Format

- **Format:** PNG
- **Style:** Terminal screenshots with syntax highlighting
  - Commands in cyan
  - Success messages in green
  - Logs in yellow
  - JSON in light blue
  - Output in light gray

## Generation

Screenshots were generated using Python PIL (Pillow) library to create terminal-style images with accurate color coding and formatting that matches typical terminal output.
