# Murphy — Collective Roadmap

## Collective Milestones (not sprints)

This roadmap is governed by the community. Priorities are set by consensus.
Each milestone is unlocked when the previous one is validated — by members,
not by MRR.

### Milestone 0 — Founding (current)
- Establish contribution model and treasury structure
- Onboard first 100 community members
- Publish Collective Charter (v0.1) for comment
- Seed the Academic Validator track with initial partner institutions

### Milestone 1 — Individual Agent Layer
- Every member gets a personal agent trained on their own workflow
- Agent = wallet + auth + identity (no separate account system)
- Contribution tracking goes live (usage → access credit)

### Milestone 2 — Social Income v1
- Community Treasury receives first inflows (paid commercial access)
- First distribution to validated contributors
- Member Validation (community consensus gate) compensated for the first time

### Milestone 3 — Collective Governance v1
- First community proposal and vote cycle
- Steward Council elections open
- Academic review board seated for model decisions

### Milestone 4 — Open Nation Infrastructure
- Murphy's services become the backbone of member-owned digital life:
  wallet, identity, credential, communication, workflow
- External governing body integration pilots (civic data, need indices)
- Interoperability with other open-source collectives
